* C/C++ 혹은 하나의 프로그래밍 언어의 기초를 다지신 분

이 문서는 완벽하지 않습니다. (최선을 다했습니다만...) 따라서 여러분들의 피드백은 언제나, 항상 환영입니다. 제가 참고한 자료들은 다음과 같습니다.

* 책 "윤성우의 열혈 자료구조"
* 책 "Thinking Data Structure"
* 인프런 강의 "권오흠의 영리한 프로그래밍을 위한 알고리즘 강좌"
